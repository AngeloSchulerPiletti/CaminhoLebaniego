Atenção, as tags são extremamente importantes para os seus usuários localizarem seus artigos através de buscas.

Atente-se para um detalhe importantíssimo. As seguintes palavras-chaves causam os mesmos matches no sistema:

  • isso, é, um, exemplo, de, palavras, chave
  • Isso é um exemplo de palavras-chave

Ou seja, Não se preocupe em separar cada palavra por vírgula, muito menos repetí-las, afinal de contas, o segundo exemplo é visualmente mais agradável ao seu leitor.

Não use mais de 800 caracteres.

Outro ponto importante é que os dois exemplo abaixo tem o mesmo resultado:

  • palavra, um pequeno exemplo, mais uma
  • palavra,um pequeno exemplo , mais uma,
