O artigo possui diversos elementos para aumentar o User Experience na leitura dos artigos. Você encontra abaixo uma lista com todos os elementos disponíveis.

A primeira vista isso pode parecer um pouco complexo, mas garanto que é muito preferível escrever @@ antes de um parágrafo à escrever: <p class="simple_p" ></p>

Lista de elementos:
Títulos:

Seus títulos devem ser escritos assim: "#Título" (sem aspas). Sendo que # é o maior título e ###### é o menor título de todos.

Estilos Inline:
  • Bold: Adicione * antes e depois do texto.
  • Itálico: Adicione ** antes e depois do texto.
Linha Horizontal:

Utilize o código "<-class class->" (sem aspas) para criar uma linha horizontal vermelha elegante. Sendo que class deve ser substituída por dois tipos de estilo:

  • Posição:
    • left
    • center
    • right
    • all
  • Estilo:
    • gorgeous
    • tiny
Listas:

Caso você tenha interesse em listar, por exemplo, itens de viagem. Você deve realizar a seguinte formatação:

Ao iniciar a lista, adicione "----".

Em seguida, para cada elemento de dentro da lista, adicione "-" antes do elemento.

Para finalizar a lista, adicione "----". Veja o exemplo abaixo:

Texto qualquer
----
-Item1
-Item2
----

Links:

Para adicionar um link no texto, utilize o código: "$Nome do link[https://www.link.com]$" (sem aspas).

O exemplo acima será exibido assim: Nome do link.

Imagens

Para saber mais sobre as imagens clique sobre o campo de imagem na criação dos artigos.

Observações importantes:

O artigo é space-sensitive, isto é, caso você deixe uma linha em branca, essa linha ficará em branco no artigo. Use para separar o conteúdo do seu artigo corretamente.
